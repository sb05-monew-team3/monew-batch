# monew-batch
> Spring Batch 기반의 자동화 데이터 관리 서버로, 뉴스 기사 수집·백업·정리 및 사용자·알림 데이터 정리를 주기저긍로 수행합니다.

---

## 개요
'monew-batch'는 MONEW 서비스의 백엔드 데이터 유지 및 관리 작업을 자동화합니다.

## 🧱기술 스택
`Java 17` `Spring Boot 3.5.6`, `Spring Batch` `PostgreSQL` `AWS S3` `AWS EC2` `Gradle` `Lombok` `SLF4J`

## ⚙️주요 기능
'monew-batch'는 Spring Batch 기반으로 구성된 job 단위의 작업 서버입니다.
각 Job은 독립적으로 실행되며, 장애 발생 시 재시작이 가능하도록 설계되어 있습니다.

### API 기반 뉴스 기사 수집
- 네이버 뉴스 API를 통해 최신 기사를 수집하고 사용자 관심사 키워드와 매칭하여 기자 저장 및 알림을 생성합ㄴ다.
### RSS 기반 뉴스 기사 수집
- 조선일보, 한국경제, 연합뉴스 등 RSS 피드를 통해 기사를 수집하고, 기사 제목의 키워드와 사용자의 관심사를 매칭하여 저장 및 알림을 생성한다.
### 기사 백업
- 기사 데이터를 AWS S3에 백업하여 데이터 유실을 방지한다.
- 기사 본문, 키워드 발행시간을 함께 저장한다.
### 알림 자동 삭제
- 생성 후 7일 이상 지난 알림을 자동 삭제하여 데이터베이스 용량과 성능을 관리한다.
### 사용자 정보 물리 삭제
- 논리 삭제된 사용자는 1일 경과 후 물리적으로 삭제하여 DB 정합성을 유지한다.
